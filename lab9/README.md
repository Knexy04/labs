
## CI/CD (GitHub Actions)
В репозитории добавлен workflow `.github/workflows/run-tests.yml`, который:
- Запускается на `push` и `pull_request`
- Ставит Python 3.11
- Устанавливает зависимости и запускает тесты для:
  - `lab5` (API-тесты против jsonplaceholder)
  - `lab6` (unit-тесты)

Шаги для использования:
1. Создайте публичный репозиторий на GitHub и запушьте код из этой папки `labs` (сохранив структуру)
2. Убедитесь, что файл `.github/workflows/run-tests.yml` присутствует в репозитории
3. Сделайте `git push` — во вкладке Actions появится запущенный workflow
4. Дождитесь зеленой галочки при успешном прогоне
